Junk Escape

  • Javascript
  • Worked with an artist
  • Game Jam entry
  • 1 week development time

Junk Escape is a game I developed in 1 week for the Community Game Jam. I collaborated with a dedicated artist who worked off of my designs for levels, enemies, and character parts. Everything from asset loading, to audio management, to the camera system was done in pure JS (as a way to learn, I have nothing against using libraries or engines). You can play it on my website

GameJ

  • Java
  • Multithreading
  • Hardware accelerated graphics
  • Documentation

GameJ is a game engine I developed slowly over the course of a year. It features separate update and rendering threads, GPU rendering, asset loading, a camera system, input handling, a debug system, and collision detection. I made it one of my goals to clearly organize and comment my code, as well as create documentation for the code base.

Master Mind

  • Java
  • Multiplayer
  • Pair Programming

A multiplayer version of the classic Mastermind board game made with Netty for networking and GameJ for the interface. The server portion was programmed together with my friend, and then we created our separate clients that connected. This video shows a game played with my client (left) and my friends client (right).
